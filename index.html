Weather App

The Weather App helps users quickly check the current weather for any city. It’s a practical project that teaches you how to:
• Fetch and display data from APIs
• Handle user input dynamically
• Manipulate the DOM using JavaScript
• Apply responsive design and styling

Core Features

• Users can type a city name and submit to see the weather.
2. Weather Display
• Shows city name and country
